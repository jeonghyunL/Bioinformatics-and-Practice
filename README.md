# Bioinformatics Term Project

## Overview
RNA-seq, RPF (Ribosome Profiling), CLIP-seq 데이터를 활용한 유전자 발현 및 번역 조절 분석 프로젝트입니다.

## Main Analysis
- **Mission 1**: 데이터 전처리 및 기본적인 발현 분석
- **Mission 2**: Start Codon 주변의 RPF (Ribosome-Protected Fragment) 분포 분석
    - Start codon 주변 5'-end 분포 확인
    - 3nt 주기성(periodicity) 관찰
    - Transcript Support Level (TSL) 1 및 Positive strand 기준 필터링

## Visualization
- Scatter plot (발현량 비교)
- Protein localization analysis
- RPF distribution around start codon
  
## Environment
- Python / Bioconda
- Tools: `samtools`, `bedtools`, `bioawk`
